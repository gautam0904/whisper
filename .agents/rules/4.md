20. Anti-Patterns (Never Do This)
    These are hard bans. A PR containing any of these is blocked regardless of context.
    console.log in production code
    any TypeScript type
    .unwrap() in Rust production code paths
    Hardcoded colour values (#fff, rgb(), hsl()) outside tokens.css
    Hardcoded pixel values outside tokens.css
    Calling invoke() directly inside a React component
    Importing from another feature's internal directory (bypass the index.ts)
    Synchronous Tauri commands that block the UI thread
    localStorage / sessionStorage (use plugin-store)
    Mutating Zustand state outside of store actions
    React.memo, useMemo, useCallback without a measured performance justification
    Disabling TypeScript strict checks with @ts-ignore (use @ts-expect-error with a comment)
    CSS !important outside of accessibility/high-contrast overrides
    Arbitrary z-index values not from a defined z-index scale
    Window.alert(), Window.confirm(), Window.prompt()
    Absolute imports that leave the src/ directory
    Magic numbers (unexplained numeric literals) in logic code
    ODOs committed to main or develop branches
    Disabling CI steps with [skip ci] except for documentation-only commits
    Do not add comments in code chages

Appendix A: Z-Index Scale
All z-index values must come from this scale. No arbitrary values.
css:root {
--z-base: 0;
--z-overlay: 100; /_ Main typing canvas _/
--z-controls: 200; /_ Bottom bar controls _/
--z-ghost: 300; /_ Ghost dot indicator _/
--z-toast: 400; /_ Error/notification toasts _/
--z-modal: 500; /_ Settings panel _/
--z-drag: 999; /_ Drag region layer _/
}

Appendix B: Shortcut Conflict Blocklist
These shortcuts must never be allowed as the Whisper summon key. Enforce with the isBlocklistedShortcut() validator.
typescriptconst BLOCKLISTED_SHORTCUTS = [
'Ctrl+C', 'Ctrl+V', 'Ctrl+X', 'Ctrl+Z', 'Ctrl+A', // Universal
'Ctrl+Alt+Delete', // Windows system
'Ctrl+Shift+Esc', // Windows Task Manager
'Cmd+Space', // macOS Spotlight
'Cmd+Tab', // macOS App Switcher
'Ctrl+Shift+S', // Zoom stop share
'Ctrl+Shift+A', // Zoom mute
'Win+L', // Windows lock screen
];
